# WelcomeHome

## 프로젝트 개요
맵 api 기반의 부동산 실거래가 조회 기능을 제공하는 Spring Boot & Vuejs 기반의 자바 웹 어플리케이션 서비스입니다.

## 프로젝트 기간 & 팀원
* 기간 : 2022. 05. 19 ~ 2022.05.26

* 팀원 : 김동욱, 박윤하
